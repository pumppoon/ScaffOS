# Admin Console

## Overview
This is the Admin Console for managing system health, configurations, and users.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd admin-console
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## API Endpoints
- **GET /api/health**: Fetches the health status of the system.
- **GET /api/config**: Fetches the configuration data.
- **GET /api/users**: Fetches the list of users.
